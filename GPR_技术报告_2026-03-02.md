## 1. 原始处理方法 (原始代码)

### 1.1 核心脚本
- `agcGain.py` - 自动增益控制
- `subtracting_average_2D.py` - 滑动平均背景抑制
- `read_file_data.py` - 数据读写与可视化

### 1.2 处理流程
```
原始CSV → readcsv(第4列) → reshape(801,79) → AGC → 背景抑制 → 输出PNG
```

### 1.3 AGC 实现
```python
# 按列(每道)归一化，窗口内最大值缩放
window = 151  # 纵向窗口
output = input / max(abs(input[i:i+window]))
```

### 1.4 背景抑制实现
```python
# 滑动道均值相减 (2D滑动平均)
ntraces = 79  # 横向窗口，覆盖全部道数
background = moving_average_2D(input, ntraces)
output = input - background
```

### 1.5 问题
- AGC 增益无上限 → 噪声过度放大
- 背景抑制窗口固定 → 适应性差
- NaN trace (第79道) 未处理 → 指标失真

## 2. 调研后的改进方法

### 2.1 文献调研结果 (15篇初筛)

| 编号 | 论文 | 核心方法 | 可实现要点 |
|-----|------|---------|-----------|
| 4 | F-K Domain Clutter Removal (2024) | F-K域阻带滤波 | 角度范围 10°-65° 抑制杂波 |
| 5 | Weighted Nuclear Norm Minimization (2023) | WNNM低秩分解 | 替代SVD，加权核范数 |
| 6 | Learned RPCA (2023) | 学习式RPCA | 减少迭代SVD开销 |
| 9 | SVD of Hankel Matrix (2019) | Hankel矩阵SVD | 滑窗SVD去噪 |
| 13 | Directional TV Minimization (2015) | DTVM方向性TV滤波 | 方向权重参数选择 |

### 2.2 复刻的改进方法

#### 改进 A: SVD 低秩背景抑制
```python
# AGC后做SVD低秩去除
U, S, Vt = svd(agc_output, full_matrices=False)
rank = 1  # 仅保留第1个奇异值 (经验值)
background = U[:, :rank] @ diag(S[:rank]) @ Vt[:rank, :]
output = agc_output - background
```

**效果**: SNR 11.64 dB (vs 基线 11.97 dB)，SSIM 0.995，耗时 0.65s (vs 1.16s)

#### 改进 B: AGC + 增益上限
```python
# 更大窗口 + 增益上限后再背景抑制
window = 301
gain_cap = 10  # 增益上限
# 先AGC限幅 → 再做subtracting_average_2D
```

**效果**: SNR 8.33 dB，SSIM 0.210 (当前参数下性能下降，需调参)

## 4. 指标定义

- **SNR**: `20*log10(RMS_target / RMS_noise)` (dB)
  - target窗口: 40×10 (y=100~700范围内搜索)
  - noise窗口: RMS最小且不与target重叠
- **对比度**: `(max_target - min_target) / (max_noise - min_noise)`
- **SSIM**: min-max归一化后全局SSIM
