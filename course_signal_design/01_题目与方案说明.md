# 现代信号处理课程设计：题目与方案说明

## 1. 题目
**基于 Hankel-SVD 与 F-K 滤波的 GPR B-scan 去噪与对比评估**

> 方向匹配：现代信号处理 + GPR（地质雷达）

## 2. 选题依据（可在周四前完成）
- **可控难度**：采用合成数据，不依赖外部实测数据采集。
- **算法成熟**：Hankel-SVD 与 F-K 滤波均为课程常见方法，资料充分。
- **可量化评估**：可直接输出 SNR、RMSE，对比清晰。
- **可扩展**：后续可替换为实测 B-scan 数据，复用同一流程。

## 3. 研究问题
在含有随机噪声、低频杂波与直达波干扰的 GPR B-scan 中，比较以下方法的去噪效果：
1. Hankel-SVD 去噪
2. F-K 滤波
3. Hankel-SVD + F-K 级联

## 4. 技术路线
1. 构建合成 GPR B-scan（双曲目标 + 层状反射）。
2. 叠加噪声与干扰（高斯噪声、低频杂波、直达波）。
3. 分别执行三种去噪流程。
4. 使用 SNR/RMSE 与图像可视化进行对比评估。
5. 给出结论与适用建议。

## 5. 预期产出
- 可运行 Python 仿真脚本（含图像与指标输出）
- 课程设计报告 Markdown（标准结构）
- PDF 导出步骤
